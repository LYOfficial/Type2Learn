# Type2Learn - Learn Everything by Typing

> Learn Everything by Typing.
>
> A desktop learning app based on typing practice.

If you like this project, please give it a star at the top right!

[![Tauri](https://img.shields.io/badge/Tauri-v2-FFC131?style=for-the-badge&logo=tauri&logoColor=white&labelColor=24C8DB)](https://tauri.app/) ![TypeScript Badge](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white) ![Rust Badge](https://img.shields.io/badge/Rust-000000?style=for-the-badge&logo=rust&logoColor=white)

**Fingers dance, knowledge enters the heart. The joy of typing, endless learning.**

Type2Learn is a tool to learn everything by typing, integrating word memorization, poetry recitation, and custom dictation, helping you deepen memory through keyboard input, whether English words, Tang and Song poems, or personal notes, all can be practiced with one click.

Currently, the software supports CET-4, CET-6, IELTS vocabulary, selected Tang and Song poems, custom TXT import, and cross-platform use on Windows/macOS/Linux, basically meeting various learning needs. The software is downloaded locally without data upload, protecting privacy and security.

The main functions of the software are three types:

- **Word Learning**——English vocabulary memorization
- **Poetry Recitation**——Poetry dictation practice
- **Custom Dictation**——Personalized content practice

During practice, you can view hints, record progress, and export data backups.

## Usage

**1 Download the release software**

Find the latest version on the project releases page: https://github.com/LYOfficial/Type2Learn/releases/latest

Download the one suitable for your system.

**2 Launch**

After launching the software, enter the main interface.

**3 Select Module**

Select "Word Learning", "Poetry Recitation", or "Custom Dictation" in the sidebar.

**4 Start Practice**

Select a vocabulary or poetry collection, click to start practice. Type in the input box according to the prompts.

**5 View Hints**

Press `Tab` key or click the light bulb button to display hints.

For more detailed instructions, please read the project documentation.

## Development

To contribute to the development and deployment of this project, please clone this repository first:

```bash
  git clone https://github.com/LYOfficial/Type2Learn.git
```

Install Rust: https://rust-lang.org/tools/install
```bash
# For MacOS users
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Start the development server:

```bash
  npm install
  npm run tauri dev
```

Build:

```bash
  npm run tauri build
```
